## What is XAUE?

XAUE turns gold into a **productive asset**. Every card is backed **1:1 by physical bullion** and powered by tokenized gold (**XAU₮ / Tether Gold**) — so the gold keeps earning yield while you hold it.

## 🎁 Products

| | |
|---|---|
| 💳 **Physical Gold Cards** | Branded, bullion-backed cards from **10g**. Bulk orders for teams & businesses (100+), produced in 7–10 days with Hong Kong shipping or same-week pickup. |
| ⚡ **Digital Gold Cards** | Instant, code-based delivery from **1 card**. Claim with a 6-digit code and a wallet connection. |

## ✨ Why XAUE

- 🥇 **1:1 physical bullion backing** — real gold behind every card

## 🔓 Open Source

XAUE Shop runs on open, verifiable infrastructure — on-chain contracts published and verified on Sourcify, plus agent-native tooling for building on top of them.

### ⛓️ On-chain contracts

| Project | Description |
|---------|-------------|
| **[xaue-agent-router](https://github.com/xaueshop/xaue-agent-router)** | Permissionless UUPS router for XAUT → XAUE mint and async redemption. |
| **[gold-token-distribution-vaults](https://github.com/xaueshop/gold-token-distribution-vaults)** | UUPS distribution vaults that pay out XAUT / XAUE for gift-card and reward flows. |

### 🤖 Agent tooling

| Project | Description |
|---------|-------------|
| **[xaueshop-mcp-server](https://github.com/xaueshop/xaueshop-mcp-server)** | MCP server exposing XAUERouter as agent tools — balances, approvals, mint, async redeem and claim. Works with Claude Code and any MCP client. |
| **[skills](https://github.com/xaueshop/skills)** | Reusable [Agent Skills](https://agentskills.io/specification) for XAUEShop operations. Install with `npx skills add xaueshop/skills`. |

Browse all repositories → [github.com/orgs/xaueshop/repositories](https://github.com/orgs/xaueshop/repositories)
